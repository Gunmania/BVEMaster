BVEMaster
=========

BVEMaster는 철도 시뮬레이션 'BVE'의 루트 데이터를 생성하는 프로그램입니다. 

콘솔 기반으로 동작하나 기본적으로 윈도우에 맞춰 만들어져있기 때문에 리눅스 등에서 사용시에는 다소 수정이 필요하며 리눅스를 지원하도록 수정 예정입니다.

연락처
* http://gunmania.kr
* d.gunmania@gmail.com

지원 기능
===
1. 일반 구문 생성
-
* 거리 입력 (미터 단위)
* 개행 (한줄 띄기)
2. Route 구문 생성
-
* 노선 설명(Route.Comment)
* 시간표 설명(Route.Timetable)
* 궤간(Route.Gauge)
* 개발자 ID(Route.DeveloperID)
3. Train 구문 생성
-
* 차량 경로 (Train.Folder)

※ 이외의 구문은 추가 예정
